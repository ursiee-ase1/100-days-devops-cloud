# Day 6: Cron Job Setup in Linux

## Objective
Learn how to schedule automated tasks using cron.

## Concepts Learned
- Cron daemon
- Crontab syntax
- Task scheduling in Linux

## Explanation
Cron allows scheduled execution of scripts or commands at fixed intervals.

## Why it matters
Automation is a core principle in DevOps. Cron jobs are used for backups, monitoring, and system maintenance.
