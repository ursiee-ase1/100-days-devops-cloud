# Day 5: SELinux Installation and Configuration

## Objective
Understand SELinux and its role in system security.

## Concepts Learned
- SELinux (Security-Enhanced Linux)
- Enforcing vs Permissive modes
- Access control beyond file permissions

## Explanation
SELinux adds an additional layer of security by controlling how processes interact with files and resources.

## Why it matters
Even if file permissions are correct, SELinux can still block access for security reasons in production systems.
