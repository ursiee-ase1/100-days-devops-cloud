# Day 10 Reflection - Blog Backup Automation

Today felt like a shift from basic Linux usage to real DevOps thinking.

At first, scripting a backup process seemed simple, but I realized how many production considerations are involved:
- Secure file transfer
- Automation without human interaction
- Proper system permissions
- Separation of local and remote storage

The most interesting part was setting up passwordless SSH. It made me understand how automation pipelines work in real environments where no one is sitting to enter passwords manually.

I’m beginning to see how DevOps is not just about tools, but about designing systems that are reliable, repeatable, and secure.

This task made me more confident in writing scripts that feel closer to real production use cases.
