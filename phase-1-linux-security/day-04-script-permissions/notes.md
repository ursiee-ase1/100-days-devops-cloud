# Day 4: Script Execution Permissions in Linux

## Objective
Understand how Linux file permissions control script execution.

## Concepts Learned
- Linux permission model (read, write, execute)
- chmod command
- Execution rights for scripts

## Explanation
In Linux, files are not executable by default. A script must be given execute permission before it can run.

## Why it matters
In production systems, incorrect permissions can prevent automation scripts, deployment scripts, and services from running.
